# Product Inventory System 

## Project Overview
This is a Django-based **Product Inventory System** developed as part of a machine test.  
The application allows an admin to manage products with full CRUD functionality, search, filtering, pagination, and low-stock indicators.

The home page displays a table of products with options to **search, filter, add, edit, and delete products**.

---

## Tech Stack
- Python 3.x
- Django 4.x
- SQLite3 (included)
- HTML, CSS (Django Templates)
